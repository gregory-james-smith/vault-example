Styelsheet template

A CSS stylesheet file is used to style the pages. This is stored in a separate repository.

The stylesheet is retrieved using the GitHub CLI and a personal access token stored as a secret to access a repo content using the GitHub API. Alternatively the GitHub API can be called with curl.

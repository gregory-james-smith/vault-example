Vault

An example

A Vault is a personal knowledge base1 consisting of a wiki hypermedia structure2 of markdown3 files.

The Vault specification aims to satisfy the design objectives. The implementation builds a Vault. This example demonstrates the features of a Vault.

  1. A database for storing and organising personal knowledge.↩︎

  2. A hypertext web publication with little inherent structure.↩︎

  3. Markdown is a lightweight markup language for rich text whose files have the .md extension and is designed to also be human readable in its plain text source code form.↩︎
